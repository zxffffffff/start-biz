# start-biz
一个快速展业的微服务脚手架，支持各类云服务 SaaS PaaS FaaS

## 架构图
![image](https://github.com/zxffffffff/start-biz/blob/main/doc/Architecture.png)

## 目标
- 快速搭建(脚手架、脚本、配置文件、...)
- 方便扩展支持各个云原生厂商(阿里云、华为云、AWS、...)
- 云计算部分支持移植到 FaaS
